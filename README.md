# 🤖 AI Sentiment Analyzer v1.0

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python&logoColor=white) ![Status](https://img.shields.io/badge/Status-Working-brightgreen)

A lightweight, rule-based Natural Language Processing (NLP) script that analyzes text and determines if the sentiment is Positive, Negative, or Neutral.

## 🧠 How It Works
Instead of using heavy neural networks, this AI uses a predefined "Dictionary Database". It splits the user's input into words, compares them with the database, and calculates a Confidence Score based on the match rate.

## 📸 Demo / Output
When you run the script, it looks like this:

```text

------------------------------

📈 Confidence Score: 90%
------------------------------

2. Download the ai_code.py file.
3. Open your terminal or command prompt.
4. Run the following command:
bash
python ai_code.py
```
